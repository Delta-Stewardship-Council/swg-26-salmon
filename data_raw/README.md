# American & Stanislaus River RST Raw Data Download

## Overview

Raw rotary screw trap (RST) data for the **American River** and **Stanislaus River** were downloaded from the [EDI Data Repository](https://portal.edirepository.org/).

---

## Download Date

`2026-04-29`
